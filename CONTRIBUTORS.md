# المساهمون

شكراً لجميع المساهمين الذين ساعدوا في تطوير SuperMarché POS!

## المطورون الرئيسيون

- **RzCode Team** - المطور الأساسي

## المساهمات

إذا كنت ترغب في المساهمة في المشروع:

1. Fork المشروع
2. إنشاء فرع للميزة الجديدة (`git checkout -b feature/AmazingFeature`)
3. إجراء التعديلات والاختبارات
4. رفع التعديلات (`git push origin feature/AmazingFeature`)
5. فتح Pull Request

## إرشادات المساهمة

- اتبع معايير الكود الموجودة
- أضف اختبارات للميزات الجديدة
- حدّث التوثيق
- اكتب رسائل commit واضحة

## أخطاء الإبلاغ

عند الإبلاغ عن خطأ، يرجى تضمين:
- وصف المشكلة
- خطوات إعادة الإنتاج
- السلوك المتوقع
- السلوك الفعلي
- معلومات النظام

## المسائل المفتوحة

تفضل بزيارة [Issues](https://github.com/rzcode/pos/issues) لرؤية المسائل المفتوحة والميزات المطلوبة.
