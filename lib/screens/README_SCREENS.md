# شاشات التطبيق

## الشاشات المنفذة ✅

### 1. SplashScreen (شاشة الاقلاع)
- شاشة الترحيب عند بدء التطبيق
- رسوم متحركة جميلة
- التحقق التلقائي من حالة تسجيل الدخول
- مدة العرض: 3 ثواني

**الملف:** `splash_screen.dart`

### 2. LoginScreen (شاشة الدخول)
- تسجيل الدخول بـ email و password
- عرض/إخفاء كلمة المرور
- رسائل خطأ واضحة
- زر دخول مع حالة التحميل
- واجهة سهلة الاستخدام

**الملف:** `login_screen.dart`

### 3. HomeScreen (الشاشة الرئيسية)
- قائمة سفلية بثلاث تبويبات
- تبويب المبيعات
- تبويب المنتجات
- تبويب الإعدادات
- عرض معلومات المستخدم

**الملف:** `home_screen.dart`

## الشاشات القادمة 🔄

### 1. SalesScreen (شاشة المبيعات)
- عرض سلة التسوق
- إضافة/حذف منتجات
- حساب الإجمالي والضرائب
- اختيار طريقة الدفع
- طباعة الفاتورة

### 2. ProductsScreen (شاشة المنتجات)
- عرض قائمة المنتجات
- البحث والتصفية
- ماسح Barcode
- معلومات تفصيلية للمنتج
- إضافة للسلة

### 3. CategoriesScreen (شاشة الفئات)
- عرض الفئات مع صور
- عداد المنتجات
- اختيار الفئة
- تصفية حسب الفئة

### 4. SettingsScreen (شاشة الإعدادات)
- معلومات المتجر
- إعدادات الضرائب
- إدارة حساب المستخدم
- تسجيل الخروج
- حول التطبيق

### 5. ReportsScreen (شاشة التقارير)
- تقارير يومية
- إحصائيات شهرية
- رسوم بيانية
- تصدير البيانات

### 6. UsersScreen (شاشة المستخدمين)
- قائمة المستخدمين
- إضافة مستخدم جديد
- تحديث المستخدم
- حذف المستخدم
- إدارة الصلاحيات

### 7. InventoryScreen (شاشة المخزون)
- عرض أرصدة المخزون
- تنبيهات المخزون المنخفض
- تسجيل الوارد والمنصرف
- تقارير المخزون

## تصميم الشاشات 🎨

### الألوان المستخدمة
- **Primary**: #1B3A5C (أزرق غامق)
- **Accent**: #E8A020 (ذهبي)
- **Success**: #16A34A (أخضر)
- **Error**: #DC2626 (أحمر)
- **Grey**: #6B7280 (رمادي)

### الفونتات
- **Headings**: Poppins (Bold)
- **Body**: Inter (Regular)
- **UI Elements**: Poppins (Medium)

### التباعد والحجم
- **XS**: 4px
- **SM**: 8px
- **MD**: 16px (الافتراضي)
- **LG**: 24px
- **XL**: 32px
- **XXL**: 48px

## الرسوم المتحركة 🎭

### في الشاشات الحالية
- Fade In/Out على شاشة الاقلاع
- Slide Up في شاشة الدخول
- Transition سلسة بين الشاشات

### للشاشات القادمة
- Page Transitions (Slide, Fade, Scale)
- List Item Animations
- Dialog Animations
- Feedback Animations (مثل الاهتزاز على النقرة)

## الحالات الخاصة 📱

### الشاشات الفارغة (Empty States)
- شاشات بدون بيانات
- رسوم توضيحية
- رسالة حث للعمل (CTA)

### حالات التحميل (Loading States)
- Skeleton Screens
- Progress Indicators
- Shimmer Effects

### حالات الأخطاء (Error States)
- رسائل خطأ واضحة
- أزرار إعادة محاولة
- تفاصيل الخطأ
